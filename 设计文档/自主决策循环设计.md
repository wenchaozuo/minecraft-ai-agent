## 一、模块定位与目标

### 1.1 现状 vs 目标

**现状（被动模式）**：
```
玩家 → 聊天 → AI解析 → 执行 → 结束
```
Bot 只是被动等待玩家指令，不主动做任何事。

**目标（自主模式）**：
```
Bot 自己决定 → 执行 → 观察结果 → 再决定 → ... → 通关
```
Bot 有自己的思考循环，持续自主决策。

### 1.2 决策循环的设计目标

```
核心功能：实现 ReAct 模式（Thought → Action → Observation）
- 每一步都有清晰的思考过程
- 可解释性强，方便调试
- 失败时有自我修正能力
```

---

## 二、ReAct 模式详解

### 2.1 什么是 ReAct

ReAct = **Re**asoning + **Act**ion

是一种让 LLM 显式推理后再行动的范式：

```
Thought: 我现在在 Y=64，目标是挖钻石。我需要先检查...
Action: navigate(position={x: 100, y: 12, z: -50})
Observation: 已到达目标位置，发现附近有钻石矿
Thought: 太好了！开始挖掘...
Action: dig(targetBlock="diamond_ore", count=3)
Observation: 成功挖掘 3 个钻石
...
```

### 2.2 为什么选 ReAct

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| ReAct | 可解释、可调试、失败易追踪 | token 消耗稍高 | **推荐** |
| 纯行动 | 简单 | 不可解释、难调试 | 简单任务 |
| 规划后执行 | 一次性规划多步 | 长距离规划效果差 | 短任务链 |

### 3.4 reactor.py - ReAct 执行器

```python
"""
ai/agent/reactor.py
ReAct 执行器 - 执行思维-行动-观察循环
"""

from typing import Optional, Dict, Any, Tuple
from dataclasses import dataclass
from enum import Enum
import json


class ActionResult(Enum):
    SUCCESS = "success"
    FAILURE = "failure"
    TIMEOUT = "timeout"
    PARTIAL = "partial"  # 部分成功


@dataclass
class ReactionStep:
    """ReAct 单步执行记录"""
    think: str
    action: Dict[str, Any]
    expected: str
    actual_result: Any
    result_type: ActionResult
   反思: Optional[str] = None


class Reactor:
    """
    ReAct 执行器
    
    负责：
    1. 构造 ReAct Prompt
    2. 调用 LLM 获取决策
    3. 解析 LLM 返回
    4. 记录执行结果
    """
    
    def __init__(self, llm_client):
        self.llm_client = llm_client
        self.history: list[ReactionStep] = []
    
    def decide(self, context: Dict[str, Any]) -> Optional[Dict[str, Any]]:
        """
        调用 LLM 获取决策
        
        Args:
            context: 包含以下键的字典
                - current_goal: 当前目标
                - world_state: 世界状态
                - recent_history: 最近行动
                - current_perception: 当前感知
                
        Returns:
            决策结果 {"type": "...", "data": {...}}
            或 None（如果无法决策）
        """
        from .prompts import REACT_SYSTEM_PROMPT
        
        # 构造 Prompt
        prompt = REACT_SYSTEM_PROMPT.format(
            current_goal=context.get('current_goal', '无'),
            world_state=context.get('world_state', '未知'),
            recent_history=context.get('recent_history', '无'),
            current_perception=context.get('current_perception', '未知'),
        )
        
        # 调用 LLM
        try:
            response = self.llm_client.chat.completions.create(
                model="deepseek-v3.2",
                messages=[
                    {"role": "system", "content": "你是一个 Minecraft AI 助手。"},
                    {"role": "user", "content": prompt}
                ],
                temperature=0.3,
                max_tokens=800
            )
            
            content = response.choices[0].message.content.strip()
            
            # 清理 markdown
            if content.startswith('```'):
                lines = content.split('\n')
                content = '\n'.join(lines[1:] if lines[0].startswith('```') else lines)
                if content.endswith('```'):
                    content = content[:-3]
                content = content.strip()
            
            # 解析 JSON
            result = json.loads(content)
            return result
            
        except json.JSONDecodeError as e:
            print(f"❌ LLM 返回格式错误: {e}")
            print(f"原始内容: {content[:200] if 'content' in dir() else 'N/A'}")
            return None
        except Exception as e:
            print(f"❌ LLM 调用失败: {e}")
            return None
    
    def execute_action(self, action: Dict[str, Any], 
                     bot_client, timeout: float = 30.0) -> Dict[str, Any]:
        """
        执行动作并返回结果
        
        Args:
            action: 动作 {"type": "...", "data": {...}}
            bot_client: Bot 客户端
            timeout: 超时时间
            
        Returns:
            执行结果
        """
        action_type = action.get('type')
        action_data = action.get('data', {})
        
        try:
            # 调用 Bot 执行
            result = bot_client.execute_action(action_type, action_data, timeout=timeout)
            return result
            
        except TimeoutError:
            return {
                'success': False,
                'type': 'timeout',
                'error': f'动作 {action_type} 执行超时'
            }
        except Exception as e:
            return {
                'success': False,
                'type': 'error',
                'error': str(e)
            }
    
    def reflect(self, action: Dict, result: Dict, 
                current_goal: str) -> Dict[str, Any]:
        """
        反思动作执行结果
        
        Args:
            action: 执行的动作
            result: 执行结果
            current_goal: 当前目标
            
        Returns:
            反思结果
        """
        from .prompts import REFLECT_PROMPT
        
        prompt = REFLECT_PROMPT.format(
            action=json.dumps(action, ensure_ascii=False),
            result=json.dumps(result, ensure_ascii=False),
            current_goal=current_goal
        )
        
        try:
            response = self.llm_client.chat.completions.create(
                model="deepseek-v3.2",
                messages=[
                    {"role": "user", "content": prompt}
                ],
                temperature=0.1,
                max_tokens=400
            )
            
            content = response.choices[0].message.content.strip()
            
            # 清理和解析
            if content.startswith('```'):
                content = '\n'.join(content.split('\n')[1:])
                if content.endswith('```'):
                    content = content[:-3]
            
            reflection = json.loads(content.strip())
            return reflection
            
        except Exception as e:
            print(f"⚠️ 反思调用失败: {e}")
            return {
                'success': result.get('success', False),
                'reason': '反思失败',
                'next_action': None,
                'should_retry': False
            }
    
    def record_step(self, step: ReactionStep):
        """记录执行步骤"""
        self.history.append(step)
    
    def get_recent_history(self, n: int = 10) -> str:
        """获取最近 N 步的格式化历史"""
        lines = []
        for i, step in enumerate(self.history[-n:], 1):
            result_icon = "✅" if step.result_type == ActionResult.SUCCESS else "❌"
            lines.append(f"{i}. {result_icon} {step.action.get('type')}: {step.think[:50]}...")
        return "\n".join(lines) if lines else "无历史记录"
```

### 3.5 agent_loop.py - 决策循环主类

```python
"""
ai/agent/agent_loop.py
自主决策循环主类
"""

import asyncio
import time
from typing import Optional, Dict, Any, Callable
from dataclasses import dataclass
from enum import Enum

from .config import CONFIG, AgentConfig
from .reactor import Reactor, ReactionStep, ActionResult
from .prompts import EMERGENCY_PROMPT


class AgentState(Enum):
    """Agent 状态"""
    IDLE = "idle"
    RUNNING = "running"
    PAUSED = "paused"
    EMERGENCY = "emergency"
    COMPLETED = "completed"
    ERROR = "error"


@dataclass
class LoopStats:
    """循环统计"""
    total_cycles: int = 0
    successful_actions: int = 0
    failed_actions: int = 0
    consecutive_failures: int = 0
    total_tokens_used: int = 0
    start_time: float = 0
    last_action_time: float = 0


class AgentLoop:
    """
    自主决策循环
    
    核心流程:
    1. 感知当前状态
    2. 获取当前目标
    3. 构建上下文
    4. 调用 Reactor 决策
    5. 执行动作
    6. 记录结果
    7. 反思（如果失败）
    8. 更新记忆
    """
    
    def __init__(
        self,
        llm_client,
        bot_client,
        planner,
        memory,
        config: AgentConfig = None
    ):
        self.llm = llm_client
        self.bot = bot_client
        self.planner = planner
        self.memory = memory
        self.config = config or CONFIG
        
        self.state = AgentState.IDLE
        self.stats = LoopStats()
        self.reactor = Reactor(llm_client)
        
        self._running = False
        self._task = None
    
    async def start(self):
        """启动决策循环"""
        if self.state == AgentState.RUNNING:
            print("⚠️ Agent 已在运行中")
            return
        
        self.state = AgentState.RUNNING
        self.stats.start_time = time.time()
        self._running = True
        
        print("🚀 Agent 决策循环启动")
        
        while self._running:
            try:
                await self.tick()
            except Exception as e:
                print(f"❌ 循环异常: {e}")
                self.state = AgentState.ERROR
                self._running = False
                break
            
            await asyncio.sleep(self.config.tick_interval)
        
        print("🛑 Agent 决策循环已停止")
    
    def stop(self):
        """停止决策循环"""
        self._running = False
        self.state = AgentState.PAUSED
        print("⏸️ Agent 已暂停")
    
    async def tick(self):
        """
        单次决策循环
        
        执行一次完整的感知-决策-执行-反思流程
        """
        self.stats.total_cycles += 1
        self.stats.last_action_time = time.time()
        
        # 1. 检查紧急情况
        if await self._check_emergency():
            return
        
        # 2. 获取当前感知
        perception = await self.bot.get_perception()
        
        # 3. 更新世界状态
        self.memory.update_world_state({
            'position': perception['environment']['position'],
            'dimension': perception['environment']['dimension'],
            'resources': perception['resources']['summary'],
        })
        
        # 4. 获取当前目标
        current_goal = self.planner.get_current_goal()
        if not current_goal:
            self.state = AgentState.COMPLETED
            print("🎉 所有目标已完成！")
            self._running = False
            return
        
        # 5. 检查目标可行性
        feasibility = self.planner.analyze_current_goal(
            perception['resources']['summary'],
            []
        )
        
        if not feasibility.can_start:
            # 目标不可行，等待或切换
            hint = self.planner.get_next_action_hint(
                perception['resources']['summary'],
                []
            )
            print(f"⏳ {hint}")
            # 可以在这里实现自动切换目标逻辑
            return
        
        # 6. 构建决策上下文
        context = {
            'current_goal': current_goal.goal.name + ": " + current_goal.goal.description,
            'world_state': self.memory.long_term.get_summary(),
            'recent_history': self.memory.short_term.get_formatted_history(
                self.config.max_history_turns
            ),
            'current_perception': perception['summary'],
        }
        
        # 7. 获取决策
        decision = self.reactor.decide(context)
        if not decision:
            print("❌ 无法获取决策，跳过本轮")
            return
        
        # 8. 提取动作
        action = decision.get('action')
        think = decision.get('think', '')
        expected = decision.get('expected_result', '')
        
        if not action:
            print("❌ LLM 未返回动作")
            return
        
        print(f"🤔 {think[:80]}...")
        print(f"   动作: {action['type']}")
        
        # 9. 执行动作
        result = await self.bot.execute_action(
            action['type'],
            action.get('data', {}),
            timeout=self.config.action_timeout
        )
        
        # 10. 记录到记忆
        self.memory.add_action(action, result)
        self.planner.queue.start_goal(current_goal.goal.id)
        
        # 11. 处理结果
        if result.get('success'):
            self.stats.successful_actions += 1
            self.stats.consecutive_failures = 0
            
            # 检查是否完成了目标
            if self._is_goal_completed(current_goal.goal, result, perception):
                self.planner.complete_current_goal()
            
        else:
            self.stats.failed_actions += 1
            self.stats.consecutive_failures += 1
            
            # 反思
            reflection = self.reactor.reflect(action, result, current_goal.goal.name)
            
            if reflection.get('should_retry'):
                # 重试
                retry_action = reflection.get('next_action')
                if retry_action:
                    await self._retry_action(retry_action)
            else:
                self.planner.fail_current_goal(result.get('error', 'Unknown'))
                
                # 如果连续失败太多次，暂停
                if self.stats.consecutive_failures >= self.config.max_consecutive_failures:
                    print(f"⚠️ 连续失败 {self.stats.consecutive_failures} 次，暂停")
                    self.state = AgentState.PAUSED
                    self._running = False
        
        # 12. 定期保存状态
        if self.stats.total_cycles % 10 == 0:
            self.memory.save()
            self.planner._save_state()
    
    async def _check_emergency(self) -> bool:
        """检查并处理紧急情况"""
        try:
            perception = await self.bot.get_perception()
            threat = perception.get('threat', {})
            
            # 判断是否紧急
            is_emergency = (
                threat.get('isDanger', False) or
                threat.get('health', 20) < 6 or
                threat.get('food', 20) < 4
            )
            
            if is_emergency:
                self.state = AgentState.EMERGENCY
                print("🚨 检测到紧急情况！")
                
                # 紧急动作
                if threat.get('food', 20) < 4:
                    action = {'type': 'useItem', 'data': {'mode': 'eat'}}
                elif threat.get('hostileCount', 0) > 0:
                    action = {'type': 'attack', 'data': {}}
                else:
                    action = {'type': 'stopAll', 'data': {}}
                
                await self.bot.execute_action(
                    action['type'],
                    action.get('data', {}),
                    timeout=10.0
                )
                
                self.state = AgentState.RUNNING
                return True
                
        except Exception as e:
            print(f"⚠️ 紧急检查失败: {e}")
        
        return False
    
    async def _retry_action(self, action: Dict):
        """重试动作"""
        print(f"🔄 重试动作: {action['type']}")
        result = await self.bot.execute_action(
            action['type'],
            action.get('data', {}),
            timeout=self.config.action_timeout
        )
        
        if result.get('success'):
            self.stats.successful_actions += 1
            self.stats.consecutive_failures = 0
        else:
            self.stats.failed_actions += 1
            self.stats.consecutive_failures += 1
    
    def _is_goal_completed(self, goal, result: Dict, perception) -> bool:
        """判断目标是否完成"""
        goal_id = goal.id
        
        # 简单的完成判断
        if 'collect' in goal_id:
            resource_key = goal_id.replace('collect_', '')
            current = perception['resources']['summary'].get(resource_key, 0)
            target = goal.provides.get(resource_key, 1)
            return current >= target
        
        # 其他目标类型...
        return False
    
    def get_status(self) -> Dict[str, Any]:
        """获取 Agent 状态"""
        current_goal = self.planner.get_current_goal()
        
        return {
            'state': self.state.value,
            'current_goal': current_goal.goal.name if current_goal else None,
            'stats': {
                'total_cycles': self.stats.total_cycles,
                'success_rate': (
                    self.stats.successful_actions / self.stats.total_cycles
                    if self.stats.total_cycles > 0 else 0
                ),
                'consecutive_failures': self.stats.consecutive_failures,
                'uptime': time.time() - self.stats.start_time if self.stats.start_time else 0,
            }
        }
```

"""

class BotClient:

    Bot 客户端
    
    负责与 JS 层的通信：
    1. 获取感知数据
    2. 执行动作
    """
    
    def __init__(self, js_server_url: str = "http://127.0.0.1:5001"):
        self.js_server = js_server_url
        self.http = httpx.AsyncClient(timeout=30.0)
    
    async def get_perception(self) -> Dict[str, Any]:
        """
        获取当前感知数据
        
        实际是从 JS 层获取，这里简化为 HTTP 调用
        """
        try:
            # 实际应该从 JS 层获取
            # 这里假设 JS 层会 POST 过来
            response = await self.http.get(f"{self.js_server}/perception")
            return response.json()
        except Exception as e:
            print(f"⚠️ 获取感知失败: {e}")
            return {}
    
    async def execute_action(self, action_type: str, 
                           action_data: Dict,
                           timeout: float = 30.0) -> Dict[str, Any]:
        """
        执行动作
        
        通过 HTTP 调用 JS 层的 action 接口
        """
        try:
            response = await self.http.post(
                f"{self.js_server}/action",
                json={
                    'type': action_type,
                    'data': action_data
                },
                timeout=timeout
            )
            return response.json()
        except httpx.TimeoutException:
            return {'success': False, 'error': 'Action timeout'}
        except Exception as e:
            return {'success': False, 'error': str(e)}
    
    async def close(self):
        await self.http.aclose()
```

## 八、视觉集成（新增）

### 8.1 视觉数据在 ReAct 上下文中的位置

视觉数据作为决策上下文的补充维度，与文本感知并行提供给 LLM：

```
决策上下文 = 
  【当前目标】      ← 来自规划器
  【世界状态】      ← 来自长期记忆
  【最近行动历史】   ← 来自短期记忆
  【当前感知（文本）】 ← 来自聚合器（精确数据）
  【当前感知（视觉）】 ← 来自 VLM 分析（空间理解）  ← 新增
```

### 8.2 LLM 的视觉交互方式

LLM 可以通过 `request_vision` 字段主动请求特定视角：

```python
# LLM 返回的决策中
{
  "think": "前方看起来像有个建筑，我需要确认一下",
  "action": {"type": "navigate", "data": {"position": {...}}},
  "expected_result": "到达建筑附近",
  "request_vision": "top_down"   # 请求俯视图
}
```

下一轮决策时，`vision_perception` 字段会包含 VLM 对该视角的分析结果。

### 8.5 视觉触发的完整流程

```
                  ┌──────────────────────┐
                  │  LLM 决策              │
                  │  返回 request_vision   │
                  └──────────┬───────────┘
                             │
                  ┌──────────▼───────────┐
                  │  AgentLoop.tick()     │
                  │  设置 vision_requested │
                  └──────────┬───────────┘
                             │ 下一轮 tick
                  ┌──────────▼───────────┐
                  │  BotClient.capture_  │
                  │  vision(camera_type) │
                  │  → JS 渲染器          │
                  │  → Python VLM 分析   │
                  └──────────┬───────────┘
                             │
                  ┌──────────▼───────────┐
                  │  vision_perception = │
                  │  { structures,        │
                  │    terrain, threats } │
                  └──────────┬───────────┘
                             │
                  ┌──────────▼───────────┐
                  │  上下文加入 vision_   │
                  │  perception → LLM    │
                  │  再次决策             │
                  └──────────────────────┘
```

### 8.6 代价控制

VLM API 调用有成本和延迟，需要合理调度：

| 策略 | 说明 |
|------|------|
| 非每轮触发 | 至少间隔 vision_interval 轮 |
| LLM 主动请求优先 | LLM 认为需要再看时才看 |
| 自动触发限频 | 洞穴等场景最多每 10 秒一次 |
| 缓存相同位置 | 位置变化 <5 格时复用上次结果 |
